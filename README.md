# 🤖 Sentiment Analysis on Social Media Posts

A deep learning based web application that analyzes the sentiment of social media posts and classifies them as **Positive 😊** or **Negative 😡** using a **Recurrent Neural Network (RNN)** built with TensorFlow/Keras.

---

## 📌 Project Overview

This project performs sentiment analysis on social media posts using **Natural Language Processing (NLP)** and a **Recurrent Neural Network (RNN)**. The application preprocesses user input, converts it into numerical sequences using a trained tokenizer, and predicts sentiment using a trained deep learning model.

A Flask-based web interface allows users to enter text and instantly receive sentiment predictions along with a confidence score.

## 📖 Working

1. User enters a social media post.
2. The text is preprocessed using NLTK (lowercasing, stopword removal, stemming).
3. The tokenizer converts text into integer sequences.
4. Sequences are padded to a fixed length.
5. The trained **Simple RNN** model predicts the sentiment.
6. The predicted sentiment and confidence score are displayed on the webpage.
